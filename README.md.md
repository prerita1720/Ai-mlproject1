# Smart Expense Tracker Application

## Project Description
The **Smart Expense Tracker** is a Python-based application that helps users manage and analyze their daily expenses.  
It allows users to:

- Add new expenses with **Date, Amount, Category, and Description**.  
- View a **summary** of total and average expenses.  
- Generate a **detailed report** showing total expenses per category along with all descriptions.  
- **Visualize expenses** with bar charts, line graphs, pie charts, and histograms.  

This project is ideal for anyone who wants to track their spending habits and make informed financial decisions.

---

## Features
1. Add expenses with full details.  
2. Summary of expenses with optional detailed view.  
3. Category-wise expense report with all descriptions.  
4. Multiple visualizations:  
   - Bar Chart (Category-wise)  
   - Line Graph (Expenses over time)  
   - Pie Chart (Category distribution)  
   - Histogram (Amount distribution)
